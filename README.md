# Welcome to Tailwind Hero!

## Hero section development using Tailwind CSS
A utility-first CSS framework for rapidly building custom designs: https://tailwindcss.com

To get started:

1. Install dependencies:

    `npm i`

2. Start dev server:

    `npm run serve`
